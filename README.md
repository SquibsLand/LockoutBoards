# Splatoon Lockout

This repository containess the configuration and icons for the Unoffical Splatoon 3 Lockout.live bingo board. This board is still in early testing and will be unbalanced.

For the latest changes regarding the beta of version lockout.live, head to [main-v2 branch](https://github.com/SquibsLand/SplatoonLockout/tree/main-v2).
